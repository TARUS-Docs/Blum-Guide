EMERGENCY: Dropped / Crashed Unit

Select the device type to view the specific crash diagnostics.

TC62 Spindle Probe

1. The "Rattle" Check (Electronics)
  1. Open battery cap. Inspect for flattened springs.
  2. Reseat batteries. Impact often breaks contact.
  3. Command a "Probe Start" (M-Code).
  4. Check RC66: If lights flash but probe stays dark, board is cracked. REPLACE PROBE.
2. The "Sticky" Test (Mechanics)
  1. Gently push stylus by hand in X, Y, Z.
  2. Listen for the "click". Visual check: does it return to absolute center?
  3. If it feels "gritty" or sticks: REPLACE PROBE.
3. The Runout Check (Alignment)
  • Bent Stem: If body runs true, just replace Stylus.
  • High Runout: Use the 4 concentricity set screws to dial it back in.
  • Cannot Adjust: If out of travel, internal shaft is bent. REPLACE PROBE.
4. Final Electronics & Latch Check (PLC)

If physical checks pass, verify the internal latch signal in Siemens Operate:

  1. Enable Probe: Go to Machine > Settings > NC > PLC Variab. Change $A_OUT[3] from 0 to 1.
    Device must blink GREEN.
  2. Monitor PLC: Watch Signal db10.dbx107.0 (Probe Feedback).
  3. Test Latch: Deflect the stylus by hand.
    • Action: The bit db10.dbx107.0 must toggle High/Low instantly.
    • Fail Condition: If the bit does not change state, the internal trigger is dead. REPLACE PROBE.

Z-Nano Tool Setter

1. The "Plunge" Check (Jamming)
  1. Action: Press the metal touch plate straight down by hand.
  2. Observation: It must move smoothly (~2mm) and snap back instantly.
  3. Fail Condition: If it sticks, returns slowly, or sits crooked: REPLACE UNIT. (Internal mechanism bent).
2. The "Crunchy" Test (Seal Breach)
  1. Action: Press the plate down and wiggle/rotate it slightly.
  2. Fail Condition: A "gritty" or "crunchy" feeling means the impact broke the IP68 seal. Coolant has entered the electronics. REPLACE UNIT.
3. The Battery Contact Check

Impacts often compress the battery springs, breaking the circuit.

  • Open the battery cap.
  • Gently stretch the springs if flattened.
  • Reseat batteries and check for lights.
4. Final Electronics & Latch Check (PLC)

If all physical checks pass, perform this final software confirmation:

  1. Enable Device: Go to Machine > Settings > NC > PLC Variab. Change $A_OUT[4] from 0 to 1.
    Device must blink GREEN.
  2. Monitor PLC: Watch Signal db10.dbx107.1 (Nano Feedback).
  3. Test Latch: Press the tool setter plate down by hand.
    • Action: The bit db10.dbx107.1 must toggle High/Low.
    • Fail Condition: If the device lights up but the bit does not flip, the electronics are damaged. REPLACE UNIT.

Pairing: Probe to IF20
REQUIRED: Blum Magnet & Pocket Flathead
  1. Set Device Type (S1 Dial):
    • Pos '1' = Touch Probe (TC62)
    • Pos '2' = Z-Nano (Tool Setter)
  2. Enter Pairing Mode: Hold SET button on IF20 until LED blinks.
  3. Trigger Probe: Hold Magnet against "SET" dot on probe. Wait for Solid Green.
  4. Save: Return S1 to '0'. Press SET once.

Troubleshooting: If the light does not turn green, you MUST turn S1 back to '0' to reset the system before trying again.

TC62 Probe: Installation & Cal
Video Tutorial (Auto-playing Muted - Tap Volume to Unmute)
Required Tools: Calibration Ring Gauge, Mag-base & Indicator (0.0001"), 0.0010" Shim Stock, Allen Wrenches, Flathead, Blum Magnet.
CRITICAL WARNING: Never tighten the small adjustment setscrews against a locked-down main screw. You will strip the threads.

Step 1: Assembly & Mechanical Zero

  1. Screw Sequence (Crucial): Locate the large flathead screw and the hex setscrew on the neck.
    • Lightly snug the flathead first.
    • Then snug the hex setscrew.
    • Note: They must be firm but not fully torqued yet. The body needs to be able to shift slightly for adjustment.
  2. Indicator Setup: Mount a 0.0001" indicator on a mag-base and sweep the stylus ball.
  3. Adjustment: Locate the 4 small setscrews at the base.
    • Loosen the screw on the "High" side slightly.
    • Tighten the screw on the "Low" side.
  4. Final Lock: Once runout is < 0.0002", fully torque the main flathead and hex neck screws to lock the assembly.

Step 2: TarusNC Calibration

Environment: TARUS NC ONLY (Siemens Operate not required)
  1. Set Surface Reference (Spindle Face Touch-off):
    • REMOVE the probe from the spindle.
    • Select "No Tool" (Empty Spindle) in the TarusNC Tool Table.
    • Using 0.0010" shim stock, jog the Spindle Face down to a flat reference surface until the shim drags.
    • Preset Z-Axis to 0.0000 (You can also set a Target value of 0.0010 to be exact).
    • Jog Z-Axis away (Retract) to a safe clearance height.
  2. Prepare Probe Tool:
    • INSTALL the probe back into the spindle.
    • Measure the probe length (Spindle Face to Stylus tip).
    • SELECT the Probe Tool in the TarusNC Tool Table (make active).
    • Enter this Actual Measured Length into the TarusNC Tool Table for the Probe Tool ID.
  3. Execute Z-Axis Calibration:
    • Jog Z axis down to approximately 1 inch (25mm) above the reference surface you just touched off on.
    • Start the Z-Calibration Cycle. The machine will probe the surface to determine tool length.
  4. Ring Calibration (XY):
    • Jog probe to the approximate center of the Ring Gauge.
    • Enter Exact Ring Diameter (etched on the ring).
    • Press Cycle Start.
Z-Nano: Tool Bump Cal
Video Tutorial (Auto-playing Muted - Tap Volume to Unmute)
Prerequisite: You need a "Precision Rod" of known length (e.g., 180.171mm).

Phase 1: Un-Hide Siemens Operate

  1. Close Apps: Use Task Manager to close TarusNC, Sproxy, and Sinumerik processes.
  2. Navigate: Open Windows Explorer > C:/Program Files (x86)/Siemens/Motion Control/siemens/sinumerik/hmi/cfg.
  3. Rename Files:
    • Rename systemconfiguration.ini to systemconfiguration-Hide.ini.
    • Rename systemconfiguration-Show.ini to systemconfiguration.ini.
  4. Relaunch: Start TarusNC. The Siemens Operate window should now be visible.

Phase 2: Data Entry & Calculation

  1. TARUS NC Tool Setup: In Tool Manager, create a tool with the EXACT length of your rod (e.g., 180.171). Make it Active.
  2. TARUS NC Position Head A-90/C0: Execute the A & C axis to A-90 C0.
  3. SIEMENS OPERATE Verify Flange Offset:
    (Minimize TarusNC to see Operate)
    Go to Operate > Machine > Setup > Channel MD > 24550[2]. Write down this value.
    Go to System Data > NC Data > Cycles > Manufacturer Cycles > BZ9953. Ensure the Flange Offset matches.
  4. SIEMENS OPERATE Manual Latch:
    • Enable $A_OUT[4] (Machine > Settings > NC > PLC Variab) -> Bump blinks GREEN.
    • Handwheel the rod carefully into the bump until it turns BLINKING RED.
    • Record the Machine Y position from the Axis Readout.
  5. Calculate Latch Position:
    Y_Latch = (Machine_Y) - (Tool_Length) - (Flange_Offset)
  6. SIEMENS OPERATE Update BZ9953: Open BZ9953 again.
    • Enter Calculated Y into Label 101 _BZN_[13].
    • Enter Eyeballed X into _BZN_[14].
    • Enter Eyeballed Z into _BZN_[15].

Phase 3: Execution

  1. SIEMENS OPERATE Set Bump Number: In Program Manager, open BLUM_CAL_A.MPF. Set _BZM_R[48] to your bump number (e.g., 1).
  2. SIEMENS + SPROXY Run Cycle: Press Cycle Start on the Sproxy Machine Panel. The machine will seek, trigger, and retract.
  3. TARUS NC Verification:
    IMPORTANT: Ensure Siemens Operate is on "Settings" softkey (or neutral screen) before running TarusNC cycle.
    • Go to TarusNC > Tools > Auto > Select Bump #1.
    • Run the routine.
    • The tool must measure within +/- 0.050mm of the rod length (180.171).

Manufacturer Troubleshooting Reference

IF20 Interface Unit
  • No Function / LEDs Off: Check 24V power supply (X1, Pin 1+2). Check polarity.
  • Outputs not switching: Check if PLC logic is Active High or Active Low (Dip switches S2).
  • Error LED On: Cable break to RC66, Short circuit in cable, or Probe Battery Low/Error.
RC66 Receiver
  • No Reception (LEDs Off): Metal chips covering receiver? Probe out of range? Probe battery dead?
  • Status LED Flashes Red: System error. Check cable X2. Check for interference from other 2.4GHz sources (Wi-Fi).
  • Interference: If signal drops, try changing the Frequency Channel (Requires Blum software/adapter, typically not field adjustable without kit).
TC62 / Z-Nano Probes
  • Non-Repeatable (Scatter): Stylus loose? Measuring speed too high? Liquid/Chips between stylus and workpiece? Machine vibration?
  • False Triggering: Vibration too high (reduce rapid speed). Stylus too heavy/long for selected trigger force?
  • No Trigger: Probe not enabled (Check $A_OUT). Signal path blocked by fixture/part.
